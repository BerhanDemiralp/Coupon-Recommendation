# Coupon-Recommendation
In-vehicle coupon recommendation project using the UCI dataset. Built end-to-end ML pipelines with preprocessing, baseline comparisons (Dummy, Logistic Regression, calibrated Linear SVC, Random Forest), Stratified K-Fold cross-validation, hyperparameter tuning, and evaluation with F1, ROC-AUC, and PR-AUC (plus ROC/PR curves and confusion matrices).
